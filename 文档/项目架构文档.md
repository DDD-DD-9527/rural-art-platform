# 乡艺未来 - 乡村艺术学习平台项目架构文档

## 项目概述

**乡艺未来**是一个全栈现代化乡村艺术学习平台，旨在传承传统文化，帮助乡村居民学习艺术技能和数字技能。项目采用前后端分离架构，集成双AI平台，提供沉浸式的学习体验。

### 完整技术栈

#### 前端技术栈
- **前端框架**: Vue 3.4.0 (Composition API)
- **构建工具**: Vite 5.0.0
- **路由管理**: Vue Router 4.2.5
- **状态管理**: Pinia 2.1.7
- **UI样式**: Tailwind CSS 3.4.0
- **图标库**: Lucide Vue Next 0.454.0 + Heroicons Vue 2.2.0
- **工具库**: VueUse 10.7.0 + Axios 1.11.0
- **开发工具**: ESLint + Prettier + PostCSS + Autoprefixer

#### 后端技术栈
- **运行环境**: Node.js 18+
- **Web框架**: Express.js 5.1.0
- **数据库**: MongoDB 6.0 + Mongoose 8.17.1
- **身份认证**: JWT (jsonwebtoken 9.0.2)
- **安全中间件**: Helmet 8.1.0 + CORS 2.8.5
- **文件上传**: Multer 2.0.2
- **日志管理**: Winston 3.17.0
- **开发工具**: Nodemon 3.1.10 + Express-validator 7.2.1

#### AI集成平台
- **广州大学智能体**: 学习助手和个性化推荐
- **Coze AI平台**: 四大创作工具（图片增强、风格转换、图案生成、智能修复）

#### 部署技术栈
- **容器化**: Docker + Docker Compose
- **反向代理**: Nginx Alpine
- **数据库**: MongoDB 6.0 容器
- **环境管理**: 开发/生产环境分离

## 核心文件详解

### 1. 配置文件层

#### `package.json`
- **功能**: 项目依赖和脚本配置
- **主要依赖**:
  - Vue 3.4.0 - 核心框架
  - Vue Router 4.2.5 - 路由管理
  - Pinia 2.1.7 - 状态管理
  - Tailwind CSS 3.4.0 - 样式框架
  - Lucide Vue Next - 图标库
- **脚本命令**:
  - `npm run dev` - 开发服务器
  - `npm run build` - 生产构建
  - `npm run preview` - 预览构建结果

#### `vite.config.js`
- **功能**: Vite构建工具配置
- **配置项**:
  - Vue插件集成
  - 路径别名 `@` 指向 `./src`
  - 开发服务器端口3000
  - 自动打开浏览器

#### `tailwind.config.js`
- **功能**: Tailwind CSS自定义配置
- **扩展内容**:
  - 自定义颜色主题（primary、accent、neutral、surface）
  - 自定义动画（float、pulse-glow）
  - 响应式断点配置

#### `postcss.config.js`
- **功能**: PostCSS处理器配置
- **插件**: Tailwind CSS + Autoprefixer

### 2. 应用入口层

#### `index.html`
- **功能**: HTML入口文件
- **特点**:
  - 设置中文语言环境
  - 响应式视口配置
  - 项目标题："乡艺未来 - 乡村艺术学习平台"
  - 挂载点 `#app`

#### `src/main.js`
- **功能**: Vue应用初始化和配置
- **主要职责**:
  - 创建Vue应用实例
  - 配置Pinia状态管理
  - 设置Vue Router路由
  - 定义21个路由（7个核心页面 + 11个扩展功能页面 + 4个AI创作工具页面）
  - 应用挂载到DOM
- **路由配置**:
  - 核心页面：首页、学习、社区、创作、课程、评估、个人资料
  - 扩展功能：成就、关注、粉丝、点赞、评论、会员、作品等
  - AI工具：图片增强、风格转换、图案生成、智能修复

#### `src/App.vue`
- **功能**: 根组件
- **结构**:
  - 全局背景渐变
  - 路由视图容器
  - AI助手浮动组件
  - 全局样式定义（玻璃拟态效果、渐变样式）

### 3. 状态管理层

#### `src/stores/user.js`
- **功能**: 用户数据状态管理（Pinia Store）
- **数据结构**:
  - `user`: 用户基本信息（姓名、头像、位置、简介）
  - `learningStats`: 学习统计（学习天数、完成课程、积分、关注者）
  - `skillProgress`: 技能进度（艺术技能、数字技能）
  - `achievements`: 成就徽章系统
  - `preferences`: 用户偏好设置
- **方法**:
  - 用户数据加载和更新
  - 技能进度更新
  - 成就徽章管理
  - 积分和连续学习天数管理
