## 故障排除

### 常见问题检查

#### 如果容器无法启动
```bash
# 查看容器日志
docker-compose -f docker-compose.simple.yml logs [service-name]

# 检查Docker服务
sudo systemctl status docker

# 检查端口占用
netstat -tlnp | grep :[port]
```

#### 如果网站无法访问
```bash
# 检查Nginx配置
nginx -t

# 检查防火墙
sudo ufw status
# 或
sudo firewall-cmd --list-all

# 检查域名解析
nslookup ai-future-rural-art.top
```

#### 如果API无法访问
```bash
# 检查后端服务
curl http://localhost:3000/api/health

# 检查数据库连接
docker exec rural-art-mongodb mongosh --eval "db.stats()"
```
