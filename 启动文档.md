# 乡村艺术学习平台启动指南

## 系统架构
- **前端**: Vue.js 3 + Vite + Element Plus
- **后端**: Node.js + Express + MongoDB
- **数据库**: MongoDB (Docker容器)

## 环境要求
- Node.js 16+
- Docker & Docker Compose
- Git

## 启动步骤

### 1. 启动数据库 (MongoDB)
```bash
# 进入项目目录
cd d:\DDD\DDD-item\cuddly-spork

# 启动MongoDB数据库 (开发环境)
docker-compose -f docker-compose.dev.yml up -d mongodb

# 或者启动生产环境数据库
docker-compose up -d mongodb
```

### 2. 启动后端服务
```bash
# 进入后端目录
cd backend

# 安装依赖 (如果还没安装)
npm install

# 启动开发服务器
npm run dev

# 或者启动生产服务器
npm start
```

### 3. 启动前端服务
```bash
# 回到项目根目录
cd ..

# 安装依赖 (如果还没安装)
npm install

# 启动开发服务器
npm run dev
```

## 服务端口
- **前端**: http://localhost:5173 (开发环境)
- **后端**: http://localhost:3001 (开发环境)
- **数据库**: mongodb://localhost:27017
